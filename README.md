# MegaExcelTablicaSozdatel

Скачать файлы из [релизов](https://github.com/Katuli2019/MegaExcelTablicaSozdatel/releases)

ВАЖНО ЗНАТЬ что если otchet.xlsx уже открыт, то программа не сработает.
